---
title: "Get Batch Status"
api: "GET https://ytrss.xyz/v1/jobs/batch/{batchId}"
description: "Get the status and details of a batch of conversion jobs"
---

## Overview

Retrieve the current status, progress, and details of all jobs in a specific batch. Use this endpoint to monitor multiple conversion jobs that were submitted together.

## Path Parameters

<ParamField path="batch_id" type="string" required>
  The unique identifier of the batch to retrieve
</ParamField>

## Response

<ResponseField name="batch_id" type="string">
  Unique identifier for the batch
</ResponseField>

<ResponseField name="finished" type="boolean">
  Have all the jobs completed
</ResponseField>

<ResponseField name="jobs" type="object[]">
  Array of job objects representing each conversion task

  <Expandable title="Job Object">
    <ResponseField name="id" type="string">
      Job ID used to poll status and download the converted content
    </ResponseField>
    <ResponseField name="url" type="string">
      The original YouTube URL being converted
    </ResponseField>
    <ResponseField name="status" type="string">
      Current job status. Will always be `CREATED` in the initial response
    </ResponseField>
    <ResponseField name="download_endpoint" type="string">
      Endpoint to download the converted file (only present when status is `SUCCESS`)
    </ResponseField>
    <ResponseField name="video_metadata" type="object">
      Video metadata (only present when status is `SUCCESS`)

      <Expandable title="video_metadata object">
        <ResponseField name="title" type="string">
          Video title
        </ResponseField>
        <ResponseField name="description" type="string">
          Video description
        </ResponseField>
        <ResponseField name="duration" type="integer">
          Video duration in seconds
        </ResponseField>
        <ResponseField name="video_id" type="string">
          YouTube video ID
        </ResponseField>
        <ResponseField name="size" type="integer">
          File size in bytes
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="counts" type="object">
  <Expandable title="counts">
    <ResponseField name="pending" type="integer">
      Number of jobs waiting to be processed
    </ResponseField>
    {" "}

    <ResponseField name="processing" type="integer">
      Number of jobs currently processing
    </ResponseField>
    {" "}

    <ResponseField name="success" type="integer">
      Number of jobs that have completed succesfully
    </ResponseField>
    <ResponseField name="error" type="integer">
      Number of jobs that failed
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl https://ytrss.xyz/v1/jobs/batch/<batchId> \
  -H "Authorization: Bearer YOUR_API_KEY"
```


```javascript JavaScript
const response = await fetch(`https://ytrss.xyz/v1/jobs/batch/${batchId}`, {
    headers: {
        Authorization: "Bearer YOUR_API_KEY",
    },
});

const data = await response.json();
console.log(data.batch);
```


```python Python
import requests

response = requests.get(
    'https://ytrss.xyz/v1/jobs/batch/{batchId}',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

batch = response.json()['batch']
print(f"Batch progress: {batch['progress']}%")
```
